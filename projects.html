FootLocker Clone

In my project, "Foot Locker Clone," I created a webpage using HTML, CSS, Bootstrap, phpMyAdmin, and MySQL. The objective was to replicate specific sections of the careers.footlocker.com website while adding extra functionality. By utilizing HTML, I ensured a well-organized and user-friendly layout, while CSS enhanced the visual appeal with custom styles. The project included a login page for accessing stored data and a registration page for users to input and store information through phpMyAdmin. I successfully duplicated three sections of the original website, maintaining its overall look and feel. Additionally, I incorporated a logout button to ensure a seamless user experience. Overall, the Foot Locker Clone project deepened my understanding of frontend design, backend database integration, and the development process as a whole.

Bagel House Calculator

The Bagel House Calculator project was developed using a combination of phpMyAdmin, Python, and GUI. This interactive calculator provided a convenient solution for customers to customize their bagel and coffee orders. With a user-friendly graphical user interface (GUI), consumers could easily select their preferred bagel, toppings, and coffee options. The calculator dynamically displayed the prices associated with each choice, allowing customers to make informed decisions. To ensure seamless data management, the project incorporated phpMyAdmin, a popular tool for managing databases. Using phpMyAdmin, client information such as order details, preferences, and pricing data were efficiently stored and organized.
